# jdbc_labs
JDBC Labs exerices

## Deployment

* Maven
At the root the project :
```
mvn install
mvn clean
mvn compile
```
Then to launch the Simple_Query class : 
```
cd target/classes/
java Simple_Query
```
To launch the Command_Line_Query class : 
```
java Command_Line_Query "your sql query"
```

## Built with

* Java
* Maven
* Mysql Connector J
